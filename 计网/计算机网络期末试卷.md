# 考试说明

考试说明：考察计算机网络全部内容（以后三章为主），难度接近真题。满分100分，共20道选择题，每题5分。请在60分钟内完成。

# 单项选择题

	B.    TCP 应答报文
	
	C.    ICMP 请求报文
	
	D.   ICMP 应答报文

2. 共有四个站进行码分多址CDMA通信.四个站的码片序列为：

	a:(-1 -1 -1 +1 +1 -1 +1 +1) b:(-1 -1 +1 -1 +1 +1 +1 -1)
	
	c:(-1 +1 -1 +1 +1 +1 -1 -1) d:(-1 +1 -1 -1 -1 -1 +1 -1)
	
	现收到这样的码片序列：（-1 +1 -3 +1 -1 -3 +1 +1），则说明（     ）发送了数据1。
	
	A.    a,d
	
	B.    b,d
	
	C.    a,c
	
	D.   b,c
