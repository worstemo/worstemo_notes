### 字符串 str —— 不可变

```
独有功能：
	大小写：upper()、lower()
	是否为数字：isdecimal()
	是否以特定字符串为开头或结尾：startswith()、endswith()
	去除字符串左右的特定字符（默认为空格）：
		strip()   左右都去除
		lstrip()  去除左边特定字符
		rstrip()  去除右边特定字符
	填充字符串到指定长度：
		center()  字符串居中
		ljust()   字符串居左，右填充
		rjust()   字符串居右，左填充
		zfill()   左填充0
	格式化输出：
		''.join()    以特定字符连接字符串
		''.format()  {}是占位符
	以特定字符为界切分字符串：split()
```

```
公共功能：
	长度：len()
	索引：str[index]
	切片：str[start:end+1:step]
	for循环遍历字符串中的每个字符
```

### 列表 list

>列表，是一个有序且可变的容器，元素可以是多种不同的数据类型

```
独有功能：
	列表尾部添加元素：append()
	特定索引前添加元素：insert()
	删除第一个匹配到的元素：remove()
	删除特定索引的元素（不写默认为尾部元素）：
		pop()
		关键字：del
```

```
公共功能：
	长度：len()
	索引：list[index]
	切片：list[start:end+1:step]
	for循环遍历列表中的每个元素
```

### 元组 tuple

>元组，是一个有序且不可变的容器，元素可以是多种不同的数据类型
>	1. 元组的元素个数不能修改
>	2. 元组的元素也不能被替换成其他的值

```
独有功能：
	无
```

```
公共功能：
	长度：len()
	索引：list[index]
	切片：list[start:end+1:step]
	for循环遍历元组中的每个元素
```
